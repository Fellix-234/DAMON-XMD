## 🚀 Quick Start

### 1. Clone Repo

```bash
git clone https://github.com/Fellix-234/DAMON-XMD.git
cd DAMON-XMD
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Bot

```bash
npm start
```
